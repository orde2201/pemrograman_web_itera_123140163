Program ini dibuat untuk memenuhi tugas praktikum pengelolaan data nilai mahasiswa. Program dapat menghitung nilai akhir, menentukan grade, menampilkan statistik, dan melakukan berbagai operasi analisis data mahasiswa.

## âœ¨ Fitur Utama

#### 1. Pengelolaan Data
- Data awal 5 mahasiswa
- Tambah mahasiswa baru
- Tampilkan semua data dalam format tabel

Tambah Buku Baru - Input interaktif untuk menambahkan buku dengan validasi lengkap
Validasi Data - Memastikan ID unik, format data benar, dan tidak ada duplikasi

#### 2. Perhitungan Nilai
- Hitung nilai akhir (30% UTS + 40% UAS + 30% Tugas)
- Tentukan grade otomatis (A, B, C, D, E)
- Hitung rata-rata nilai kelas

Pencarian berdasarkan ID - Cari item spesifik dengan ID unik
Pencarian berdasarkan Judul - Mendukung partial match (case-insensitive)

#### 3. Analisis Data
- Cari mahasiswa dengan nilai tertinggi
- Cari mahasiswa dengan nilai terendah
- Filter mahasiswa berdasarkan grade
- Tampilkan statistik lengkap kelas

#### 4. Fitur Tambahan
- Validasi input nilai (0-100)
- Distribusi grade
- Menu interaktif
- Error handling
